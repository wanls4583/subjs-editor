# subjs-editor

## 功能特性
* [x] 使用原生 JavaScript 开发，可在浏览器中运行
* [x] 支持 JavaScript 代码的高亮，折叠，复制，删除，撤销，重做等操作
* [x] 支持百万行级别的代码
* [x] 可扩展其他语言高亮
